# Playgama Bridge Claude Plugins

A marketplace of Claude Code plugins for [Playgama Bridge](https://github.com/playgama/bridge).

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add playgama/bridge-claude-plugins
```

Or for local development:

```bash
/plugin marketplace add ./bridge-claude-plugins
```

## Installing Plugins

List available plugins:

```bash
/plugin marketplace list
```

Install a plugin:

```bash
/plugin install plugin-name@bridge-claude-plugins
```
